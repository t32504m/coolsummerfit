# FIT — ニュアンスカラー試着室（モック）

ジャケット × ポロ × スラックスの配色を組み替えて試せる、コーディネート検討用のモックアプリです。
単一の `index.html` のみで動作します（ビルド不要・外部依存は Web フォントのみ）。

**デモ:** GitHub Pages を有効化すると `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

## 機能
- **3パーツ × 5色を選択** — ジャケット／ポロ／スラックスをタップで切り替え（各色に HEX とメモ付き）
- **テーマ切り替え** — Cool Summer / Autumn Earth / Monotone Chic の3パレット
- **襟の種類** — ボタンダウン／レギュラー／バンドカラー
- **袖丈** — 長袖／半袖（シャッフル時はロック）
- **シャッフル** — 色をランダムに着地（袖丈は固定）
- **お気に入り保存** — 組み合わせを保存してワンタップで呼び戻し
- **モバイル最適化** — 画面上部にプレビューを固定し、色を選びながら着姿を確認できる

## 公開手順（GitHub Pages）
1. このリポジトリの Settings → Pages を開く
2. Source を `Deploy from a branch`、Branch を `main` / `/(root)` に設定
3. 数十秒後に上記 URL で公開

## ステータス
配色検討用のモック（プロトタイプ）です。着姿は簡易イラストで、実際の製品画像ではありません。
